## Setup 
This project aims to create a Voice Assistant with the help of Python and its libraries. It is named as "Jarvis 1.0" and it is able to interact with the users and give response to user by searching on web, chatgpt, weather etc. Various libraries have been used in this project such as wolframalpha, openai, speech recognition, pyjokes, json, random, datetime and many more.
